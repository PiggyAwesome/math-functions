# math-functions
Useful and easy to use math functions!

### Greater than Function
+ x > y; returns 1 or 0

$$
g(x, y) = \frac{|x-y| + x-y}{2(x-y)}
$$

### Pythagoras Function
+ input 0 into the variable that you want to solve 

$$
p(a, b, c) = \sqrt{|b^2 + c^2|} * g(1, a) + \sqrt{|a^2 - c^2|} * g(1, b) + \sqrt{|a^2 - b^2|} * g(1, c)
$$
